# Image Captioning using CNN and LSTM

## Project Overview
This project generates captions for images automatically using deep learning. It uses CNN for image feature extraction and LSTM for caption generation.

## Objective
To build an AI model that can understand an image and generate a meaningful caption.

## Technologies Used
- Python
- TensorFlow / Keras
- CNN
- LSTM
- MobileNetV2
- Google Colab

## Project Workflow
1. Load image-caption dataset
2. Extract image features using CNN
3. Preprocess captions
4. Train CNN + LSTM model
5. Generate caption for an image
